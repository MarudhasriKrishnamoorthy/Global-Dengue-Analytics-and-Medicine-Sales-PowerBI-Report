# Global-Dengue-Analytics-and-Medicine-Sales-PowerBI-Report
Dengue Cases in the World – Interactive Power BI Dashboard featuring KPI cards, MTD/YTD/Inception analysis, synchronized slicers, bookmarks, navigation buttons, country-wise trends, gender distribution, and medicine sales insights. Built using Power BI, Power Query, DAX, and data modeling
# Dengue Cases in the World – Power BI Dashboard

## Project Overview
This Power BI project provides an interactive analysis of global dengue cases and medicine sales data through a user-friendly dashboard.

## Features
- KPI Cards (Total Cases, Active Cases, Critical Cases, Deaths)
- MTD, YTD, and Inception Analysis
- Gender Distribution Analysis
- Top 6 Countries Visualization
- Monthly & Cumulative Trends
- Medicine Sales Dashboard
- Synced Slicers Across Pages
- Interactive Navigation Buttons

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Dashboard Pages
### Page 1: Dengue Cases Analysis
- Case KPIs
- Gender Distribution
- Country Analysis
- Monthly Trends

### Page 2: Medicine Sales Analysis
- Product-wise Sales
- Country-wise Sales
- Fiscal Month Trends
